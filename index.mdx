# Panda ETL API Documentation

Welcome to the Panda ETL API documentation. Our API provides powerful tools for document processing, text extraction, and structured data retrieval from various document formats.

## Key Features

- **Parse**: Extract text content from various document formats, including PDFs
- **Extract Fields**: Convert raw document content into organized, machine-readable data
- **User Usage**: Monitor your API usage and credit consumption

## Authentication

All API requests require authentication using an API key. Include your API key in the `x-authorization` header of each request:

```
x-authorization: Bearer your_api_key_here
```

## Base URL

All API endpoints should be prefixed with the following base URL:

```
https://api.panda-etl.ai/v1
```
